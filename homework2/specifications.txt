Spécifications

insert t: tree :
@Pre: ok() => C'est un 2-3 tree qui est bien ordonné
@Post: ok() (même chose que pour la pré)
       && elements(t') == elements(t) + multiset(i)
       (elements est une fonction qui retourne un multiset de tous les éléments de l'arbre)
       En gros, après le insert, on a les mêmes éléments + celui qui vient d'être ajouté (i)


join(t1: Tree, t2: Tree) returns (t: Tree)
@Pre: ok(t1) && ok(t2)
@Post: ok(t) && elements(t) == elements(t1) + elements(t2)
